# UserImporter

This is just a full feature component that uses the `UserItem`, `UserList`,
`DropCVSJSONFile` components. Using this component you'll be able to upload
`CSV` and `JSON` files and preview this data on your browser. It can also be
used to fetch and show data on an external endpoint.
