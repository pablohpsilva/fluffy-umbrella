# UserList

This component will display a list of user related data. Internally it uses
the `UserItem` component. When `UserItem` are available, clicking on it
will show all data available.

## Props

| Name  | Description                       | Type        | default       |
| ----- | --------------------------------- | ----------- | ------------- |
| user2 | A list of users data for sampling | `User[]`    | `[{...},...]` |
| style | CSS style extension               | `CSS Style` | `{}`          |

### Users prop

This is an array of the following User object. The properties are shown below.

| Name      | Description | Type     | default |
| --------- | ----------- | -------- | ------- |
| title     | Title       | `string` | `''`    |
| lastName  | Last name   | `string` | `''`    |
| firstName | First name  | `string` | `''`    |
| email     | Email       | `string` | `''`    |
| picture   | Picture     | `string` | `''`    |
| userId    | UserId      | `string` | `''`    |
| login     | Login       | `string` | `''`    |
| password  | Password    | `string` | `''`    |
| gender    | Gender      | `string` | `''`    |
| URL       | URL         | `string` | `''`    |
| address   | Address     | `string` | `''`    |
